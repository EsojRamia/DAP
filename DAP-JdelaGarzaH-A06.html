Getter y Setter

Instead of defining separate methods as getters and setters, jQuery uses the same method as both a getter and a setter and determines which action is being performed by what parameters are passed to the method. When used as a setter the method will set values on every element within the jQuery object and then return the jQuery object allowing the method to be used as a setter anywhere within a method chain. When used as a getter the method returns the value from the first element in the jQuery object and therefore must be the last method in the chain.

Comando Explicacion Ejemplo
jQuery.get() Load data from the server using a HTTP GET request.

$.get( "test.php", function( data ) {
alert( "Data Loaded: " + data );
});

addClass() Adds one or more classes to the selected elements

$("button").click(function(){
$("h1, h2, p").addClass("blue");
$("div").addClass("important");
});

removeClass() Removes one or more classes from the selected elements

$("button").click(function(){
$("h1, h2, p").removeClass("blue");
});

toggleClass() Toggles between adding/removing classes from the selected elements

$("button").click(function(){
$("h1, h2, p").toggleClass("blue");
});

attr() get or set values on HTML attributes.

var rad, tx;
rad = document.getElementById('r1');
tx = 'The value of the name attribute of the first radio button is "' + rad.name + '"';
document.getElementById('txt').innerHTML = tx;

css() Sets or returns the style attribute

$("p").css({"background-color": "yellow", "font-size": "200%"});

val() to get or set form field values.

var singleValues = $( "#single" ).val();

html() to get or set the HTML content.

$( "p" ).click(function() {
var htmlString = $( this ).html();
$( this ).text( htmlString );
});

text() to get or set plain text content.

var str = $( "p:first" ).text();
